creamos un proyecto en angular 16.2.12
	*ng g new 
instalar capacitor
para la conexion
	npm install @capacitor/core@5.2.2 @capacitor/cli@5.2.2
	npx cap --version

agregar andoriod
	npm install @capacitor/android@5.2.2 --instalar esa version de capacitor que es la compatible 
	npx cap add android --------------------generar el archivo para android
	ng build
	ionic build                    -------- para generear el archivo para una app en android  cual quiera de las 2

para  abrir en androdstudio
export CAPACITOR_ANDROID_STUDIO_PATH=/snap/android-studio/current/bin/studio.sh---cuando no encuentra android studio
	npx cap sync android -------------------sincronyzar con android
	npx cap open android
